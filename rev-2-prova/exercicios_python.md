## 1. Academia (Conjuntos)

Uma academia oferece aulas de **Musculação** e **Funcional**. Os alunos
matriculados em cada modalidade estão armazenados nos conjuntos abaixo:

``` python
musculacao = {"Ana", "Bruno", "Carlos", "Daniela"}
funcional = {"Carlos", "Eduarda", "Fernanda", "Bruno"}
```

Desenvolva o código para criar um dicionário atribuindo os valores
solicitados abaixo. Para obter os valores, use apenas conjuntos.

-   a)  Mostrar os alunos matriculados nas duas modalidades.
-   b)  Mostrar os alunos que fazem somente Musculação.
-   c)  Mostrar os alunos que fazem somente Funcional.
-   d)  Mostrar todos os alunos únicos cadastrados na academia.

## 2. Loja Virtual (Dicionário + Tuplas)

Uma loja virtual armazena informações de produtos em um dicionário. Cada
item refere-se a uma tupla e contém nome, preço e quantidade em estoque.

``` python
produtos = {
    "p1": ("Teclado", 120.0, 5),
    "p2": ("Mouse", 80.0, 10),
    "p3": ("Monitor", 900.0, 2)
}
```

Desenvolva um programa em Python que:

-   a)  Calcule o valor total de todos os produtos em estoque.\

-   b)  Mostre o produto com maior valor unitário.\

-   c)  Exiba a soma total das quantidades de produtos.\

-   d)  Mostre uma lista contendo apenas os nomes dos produtos.

## 3. Salários (Lista)

Uma empresa armazena os salários dos funcionários em uma lista:

``` python
salarios = [1800, 2500, 3200, 4100, 2900]
```

Desenvolva um programa em Python que:

-   a)  Exiba o primeiro e o último salário da lista, após ordená-la de
        maneira crescente.\

-   b)  Calcule a média salarial.\

-   c)  Mostre o maior salário cadastrado.\

-   d)  Crie uma nova lista contendo apenas salários acima de 3000.

## 4. Biblioteca (Dicionário)

Um sistema de biblioteca armazena informações de livros em um dicionário
no formato:

``` python
livros = {
    "Livro A": 5,
    "Livro B": 2,
    "Livro C": 0
}
```

Onde o valor representa a quantidade disponível no estoque.

Desenvolva um programa em Python que:

-   a)  Mostre todos os livros cadastrados.\

-   b)  Verifique se o livro `"Livro C"` está disponível para
        empréstimo.\

-   c)  Calcule o total de livros disponíveis no estoque.\

-   d)  Exiba apenas os livros com quantidade maior que zero.
